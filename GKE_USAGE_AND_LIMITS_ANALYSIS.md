### **Alert Configuration**
```yaml
# Resource utilization alerts
- alert: HighCPUUsage
  expr: container_cpu_usage_seconds_total > 0.8
  for: 5m
  labels:
    severity: warning
  annotations:
    summary: High CPU usage detected

- alert: HighMemoryUsage
  expr: container_memory_usage_bytes > 0.8
  for: 5m
  labels:
    severity: warning
  annotations:
    summary: High memory usage detected
```
